# dracarys-chain

Installation
------------

```xml
<dependency>
    <groupId>huh.enterprises</groupId>
    <artifactId>dracarys-chain</artifactId>
    <version>1.0.0</version>
</dependency>
```

Usage
------

```java
@EnableDracarysChain
@SpringBootApplication
public class Application {
	
}

```
